☢️ RBMK-1000 Simulator

A mobile-friendly simulator inspired by the control room of the RBMK-1000 reactor.

📸 Overview

RBMK-1000 Simulator is an HTML, CSS and JavaScript project that recreates the operation of an RBMK reactor control panel. The simulator includes power management, control rods, water pumps, alarms, random events and a sandbox mode.

✨ Features

- ☢️ Reactor power control
- 🌡 Temperature monitoring
- 💧 Water flow management
- ⚡ Electrical power generation
- 🎛 Control rod insertion and removal
- 🚨 Emergency SCRAM button
- 🔊 Alarm system
- 🎲 Random events
- 🧪 Sandbox mode
- 📈 Real-time power graph
- 📱 Mobile-friendly interface
- 🟢 Soviet-inspired control room design
